# CUHKSZ 课程资料

如果遇到无法在线预览的文件，可以下载后查看。

资料下载页：https://dsanying.github.io/CUHKSZ_sourse/

## 文件夹分类

每个课程目录下按资料用途分类：

- `学习资料`：教材、参考书、课件、笔记、课程大纲等。
- `考试资料`：期中、期末、样题、答案、复习资料、允许考试携带的 cheating paper 等。
- `作业习题`：作业、习题、题解、题库。
- `其他资料`：体育评分标准、校园资料等不属于以上类型的文件。

## 命名规范

- 文件名以课程代码开头，例如 `CSC3001_2023_Midterm.pdf`。
- 同一文件夹内需要排序的内容使用补零数字，例如 `CEC1000_01_第一章知识点梳理.docx`。
- 考试资料使用 `课程_年份[_Summer][_Sample]_Midterm/Final[_WithSolution].pdf`。
- 文件名尽量只写标题和必要版次，不重复写资料类型。
- 同一份资料只保留一份；若内容相同，优先保留更清晰、更干净的版本。
